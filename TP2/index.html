Les listes

Il existe les listes ordonnées, les listes non-ordonnées et les listes de définitions

Les listes ordonnées

Les listes ordonnées en HTML sont créées en utilisant la balise ol. Voici un exemple de code HTML pour créer une liste ordonnée :

  1. Element 1
  2. Element 2
  3. Element 3

Dans cet exemple, la balise ol est utilisée pour encadrer les éléments de la liste ordonnée, et chaque élément de la liste est créé en utilisant la balise li. Le numéro d'ordre sera généré automatiquement par le navigateur web. Il est également possible de spécifier un numéro de départ pour la liste en utilisant l'attribut start. Par exemple, si vous souhaitez commencer votre liste à partir de 5 plutôt que de 1, vous pouvez utiliser la syntaxe suivante :

  1. element1
  2. element2
  3. element3

Dans ce cas, les numéros de la liste commenceront à 5 au lieu de 1.
